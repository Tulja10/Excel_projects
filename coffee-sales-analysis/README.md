# Coffee Sales Dashboard 

## Overview

This project analyzes the sales performance of a coffee business using an interactive Excel dashboard.
The goal is to understand **sales trends, customer behavior, and product performance** to support better decision-making.

The dashboard allows users to explore data dynamically using slicers and a timeline.


## Dataset

The dataset consists of three tables:

* **Orders** → Contains order date, quantity, and sales amount
* **Products** → Includes coffee type, roast type, and size
* **Customers** → Contains customer details and loyalty status

These tables were cleaned and combined to create meaningful insights.


## Key Features

* **Timeline Filter** → Analyze sales across different time periods

* **Slicers** → Filter by:

  * Roast Type
  * Coffee Type
  * Size
  * Loyalty Card

* **Charts Included**:

  * Sales Trend Over Time
  * Sales by Country
  * Top 5 Customers

* **KPIs**:

  * Total Sales
  * Total Orders
  * Average Order Value
  * Top Coffee Type


## Key Insights

* **Excelsa is the top-performing coffee type**, generating the highest sales.
* **United States contributes the majority of revenue**, significantly higher than other countries.
* Sales show **fluctuations over time**, indicating seasonal or demand-based variation.
* A small group of customers contributes a **large portion of total sales**.
* Order value remains relatively stable, indicating **consistent customer spending behavior**.


## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers & Timeline
* Basic formulas (GETPIVOTDATA, aggregation)
